# ShopSphere

A full-stack e-commerce web application that allows users to browse products, manage carts, place orders, submit reviews, and track purchases through an interactive shopping platform.

## Live Demo

**Front-end:** https://shopsphere-althafch.netlify.app/login

**Back-end:** https://shopsphere-production-963f.up.railway.app/

---

## 👨‍💻 About

ShopSphere is a modern full-stack e-commerce platform developed by Mohammed Althaf C H as part of a Full Stack Development Internship.

The application enables users to register, log in securely, browse products across multiple categories, add items to their cart, place orders through a simulated payment gateway, submit product reviews, and track order status. An administrator dashboard is included for product and order management.

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm
* Modern Web Browser

### Installation

Clone the repository:

```bash
git clone https://github.com/Althaf-CH/shopsphere.git
```

Navigate into backend:

```bash
cd ShopSphere/back-end
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npm start
```

Server runs on:

```text
http://localhost:5000
```

Open frontend files using Live Server or a browser.
